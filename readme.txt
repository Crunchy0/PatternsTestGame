Обозначения:

"X " - стена
"I " - входная точка
"O " - точка выхода
"/\" - игрок
"YY" - враги

Управление:

"Q" - выход
"W", "S" - идти вперёд / назад соответственно
"A", "D" - поворот налево / направо (поворот постепенный, задаёт направление взгляда игрока)
"K" - атака (уменьшает здоровье врагов)
"O", "P" - сохранить / загрузить последнюю игру
